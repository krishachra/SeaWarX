# 🌊⚓ SeaWarX – Battleship Game (React)

> A modern, interactive Battleship Game built using React.js with dynamic grid logic, real-time gameplay, and an AI-based opponent.

---

## 📌 Overview

Battleship Game is a web-based strategy game where a player competes against a computer opponent on a grid. The objective is to destroy all enemy ships before your own fleet is sunk.

This project recreates the classic naval battle experience with a clean UI and smooth gameplay using React.

---

## 🎯 Features

- Player vs Computer gameplay  
- Interactive ship placement  
- Randomized computer ship generation  
- Turn-based attack system  
- Hit 🔥 / Miss ❌ indicators  
- Winner detection system  
- Responsive design  
- Modular React components  

---

## 🧠 Tech Stack

- ⚛️ React.js  
- 🟨 JavaScript (ES6+)  
- 🎨 HTML5 & CSS3  

---

## 🕹️ How to Play

1. Place your ships on the grid  
2. Click on the enemy grid to attack  
3. 🔥 = Hit | ❌ = Miss  
4. Computer attacks automatically  
5. Sink all enemy ships to win
